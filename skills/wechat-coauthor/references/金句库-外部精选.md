## 已确认入库（用户逐条选定）

### 001 · 成功了还在怀疑

**原句**（黄叔）：「看着月底收入的表格，还是会怀疑说，这个月我做得对么？」

**句式抽象**：
```
看着 [具体的、可见的物件]，
还是会 [怀疑/问自己] [一个没有答案的具体问题]
```

**核心**：**具体物件 + 具体问题**。两个"具体"缺一不可。
- "月底收入的表格" = 读者能想象你坐在电脑前的画面
- "这个月我做得对么" = 没有答案，不是修辞，是真问题

**意疏可用场景**：
- 写某篇文章发出去后的感受
- 写某个决定做完之后的状态
- 任何"事情结束了但还没结束"的时刻

**用法示例**：
- "文章发出去了，关掉后台的时候还是会想，这个选题是不是不够准"
- "看着昨天那篇的阅读量，有点开心，但还是不确定那样写是不是对的"

**禁止原句直接用**：是黄叔的具体财务场景，不是意疏的。

### 002 · 时间锚定反应

**原句**（黄叔）：「只震惊了 5 秒，然后就开始思考」

**句式抽象**：
```
只/才 [数字] 秒/分钟，然后 [具体的下一个动作]
```

**核心**：**数字要小，动作要具体**。
- 数字小 = 反应快，但不是"我很厉害"式表扬，是时间线描述
- 动作具体 = "开始思考"比"思考了"动感，但更好是"打开备忘录"

**意疏可用场景**：
- 遇到一个新工具/新消息，写自己的第一反应
- 某件事打断了你，写你怎么处理

**用法示例**：
- "愣了大概 3 秒，截图，存到 AI 口袋的素材文件夹"
- "看到这条消息，停了一下，大概 10 秒，然后关掉了"
- "5 分钟之后我回过神来，把那篇文章又从头看了一遍"

**禁止原句直接用**：数字和语境都是黄叔的具体情境。
