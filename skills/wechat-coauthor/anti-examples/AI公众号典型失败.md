# Anti-example:典型 AI 公众号失败稿(逐句解剖)

> 这是用户给意疏 skill 反馈时,AI 实际产出的一篇关于 Hermes Agent 的稿子。**不是个例,是 AI 公众号写作的"默认形态"**。
>
> 用法:写完稿子拿不准是否掉进默认套路时,对照这里看。**典型失败长这样,看完反向操作即可**。

---

## 失败稿原文(节选)

```
Agent 这个词快被讲烂了,直到我用上 Hermes

这两年,Agent 这个词真的快被讲烂了。
今天一个「自主执行」。明天一个「长期记忆」。
后天一个「你的 AI 员工」。
听起来都很猛。
但很多时候,用起来还是老样子。

你问一句,它答一句。
最多帮你写段代码,改个文案,生成一个表格。
有用吗?有用。
但离「替我干活」,总觉得还差一口气。

直到我这几天开始折腾 Hermes。

我才发现,Agent 这东西如果只待在网页里,就很容易变成一个高级聊天框。
但它一旦进了你的微信、飞书、电脑和文件系统,味道就完全不一样了。
```

---

## 逐项诊断(对照 SKILL.md 红线)

### ❌ 红线 1:结构反模板 / 开场禁用清单

**违反**:第一句"这两年,Agent 这个词真的快被讲烂了"——是 SKILL.md 明令禁用的"X 这个词被讲烂了"句式。

**意疏会怎么开**:
> "前天晚上 11 点,我盯着电脑屏幕,Hermes 正在帮我整理一个微信群最近一周的发言。我看着光标一行行跳,心里咯噔一下——这玩意儿,真的钻进微信了。"

**差别**:具体时间戳 + 在场感 + 情绪锚点。

### ❌ 红线 2:voice 锚点密度 = 0

**违反**:全文 1500+ 字,**没有 1 处真态度词**。出现频率最高的句子是:
- "听起来都很猛"——评价但不是态度
- "有用吗?有用"——伪反问
- "感觉会明显不一样"——伪体验
- "这就够了"——廉价升华

**意疏的态度长什么样**(对照卡兹克范文):
- "我现在就仿佛鸡排哥"
- "我心里咯噔一下"
- "真赤鸡"
- "贵死了"

这类词不是 keyword,是**作者承认自己有具体情绪**的标志。AI 失败稿里完全缺失。

### ❌ 红线 4:节奏机械

**违反**:开头 5 段全是短句堆叠,每段 1-2 句。

```
你问一句,它答一句。
最多帮你写段代码,改个文案,生成一个表格。
有用吗?有用。
但离「替我干活」,总觉得还差一口气。
```

这是**典型的伪卡兹克节奏**——只学到了"短段落",没学到"短段落是情绪段,论证段还是要有完整推理"。结果整篇变成节拍器,没有呼吸。

### ❌❌❌ 红线 5:句式级 AI 味——全部命中

这篇文章命中了红线 5 的 5 种 AI 句式中的 **5/5**:

| 红线 5 子项 | 失败稿原文 |
| --- | --- |
| 假反应编排 | "我看到这儿,第一反应不是「哇,好厉害」。是:坏了。" |
| 伪深刻凭空概念 | "它改变的是你使用 AI 的姿势" / "味道就完全不一样了" |
| 万能转折 | "总觉得还差一口气" |
| 哲学感升华 | "这两个东西不是一回事" / "这就够了" / "这个方向是真的能跑起来" |
| 标语收尾 | "把看见的机会,变成撑起自己的能力" |
| 万能反思句式 | "Agent 这东西如果只待在网页里,就很容易变成一个高级聊天框" |

### ❌ 红线 6:删签名测试 —— 完全失败

把结尾签名删掉,这篇文章**可以发到任何一个 AI 自媒体号上**:数字生命卡兹克、AI 头条、量子位、机器之能...谁的号都能发,**没有一处指纹**指向「意疏的AI口袋」。

这是**最致命的失败**。意疏 voice 的核心就是不可替代性。

### ❌ 虚构嫌疑

文章中段写"我最开始注意到 Hermes,是因为刷到 意疏 在用它管微信。他问了一句:某个群意疏最近一周都说过啥?"

**意疏自己有没有说过这句话?** 如果是 AI 凭空编出来的"刷到意疏..."的场景——**这是严重违规**,违反了 SKILL.md 红线 3 的"具体优先于抽象但绝不编造"。

写之前必须问用户:"这个场景是你真实经历过的吗?" 不是 → 标 `[在这里插入你的真实例子]` 让用户填,**绝不替用户编**。

## 这篇为什么失败的根本原因

**不是 AI"写得不够好"——是 AI 套用了它见过的"AI 公众号文章模板"**。

这个模板长这样:

```
1. 开篇:吐槽行业现状("X 词被讲烂了" / "都在卷 Y") 
2. 转折:亮出主角("直到我用上 Z")
3. 情景演示:简单步骤 + 个人感慨
4. 安全提醒(必须有,显得负责任)
5. 升华:用一句标语收束
6. 引流:"关注 XX,我们一起..."
```

**这个模板任何 AI 公众号都在用**。Claude 默认调用这个模板,是因为训练数据里这种文章铺天盖地。

**SKILL.md 的工作不是让 Claude"按规则写",是让 Claude 在写之前先**意识到自己正在调用这个默认模板**,然后主动覆盖**。

---

## 如何用这个 anti-example 做诊断

写完文章后,把自己的稿子和这篇放一起对比:

1. 我的开头和这篇的开头,**结构上是不是一样**?(都是"行业吐槽+主角登场"?)
2. 我有几处真态度词?这篇是 0,**我有几个?**
3. 我的段落节奏,是不是也是 1-2 句一段堆 5 段?
4. 我的结尾,是不是也用了"这就够了 / X 不是 Y"这种廉价反思句?
5. 我的"删签名测试"通过了吗?

任一条命中 → **不是修改,是重写**。

---

## 一句话总结

意疏要的不是"更好的 AI 公众号文章",是"**不像 AI 写的公众号文章**"。差别比看起来大。
