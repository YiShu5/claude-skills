# 中英文混排规则(公众号强制)

公众号读者在手机上看,混排不规范**非常显眼**。

## 4 条核心规则

1. 中文 + **空格** + 英文 / 数字
2. 英文 / 数字 + **空格** + 中文
3. 英文 / 数字 + 标点(**无空格**)+ 中文
4. 中文 + 标点(**无空格**)+ 英文 / 数字

## 标点全部用全角

- 逗号 = `,`(不是 `,`)
- 句号 = `。`(不是 `.`)
- 冒号 = `:`(不是 `:`)
- 引号 = `「」`(不是 `""` 或 `''`)
- 双重引号 = `『「」』`

## 示例

✅ 正确:
- 「我用 Cursor 做了一个项目」
- 「AI 功能使用量涨了 10 倍」
- 「获得了 5000 美金(约 3.5 万人民币)」
- 『这是「AI 友好」的文档』

❌ 错误:
- 「我用Cursor做了一个项目」(中英文之间缺空格)
- 「我用 Cursor 、 Claude Code」(顿号前后多余空格)
- 「获得了 5000 美金,约 3.5 万人民币」(英文逗号)

## 自检清单

- [ ] 中文与英文之间有空格
- [ ] 中文与数字之间有空格
- [ ] 标点符号前后无多余空格
- [ ] 全文使用中文全角标点
