# 图片 Prompt 规范（封面 + 文中配图）

> SKILL.md 写图片 prompt 时必读。所有封面与配图都按本文件的五要素结构生成。

## 五要素结构（强制顺序，中文逗号分隔）

1. **核心意象**：从文章前 200 字提取 2-4 个具象名词（人/物/场景/动作）。**不要抽象概念**（"效率""成长""创新"都不算）。
2. **情绪基调**：恰好 1 词，从下表七选一——
   `温暖治愈` / `专业冷静` / `紧张焦虑` / `轻松愉悦` / `怀旧沉思` / `励志昂扬` / `神秘高级`
3. **画面风格**：按文章主题查下表选 1。
4. **构图与镜头**（固定文本，原样拼接）：
   `2.35:1 横向构图，主体居中偏左，右上角嵌入小字「意疏的AI口袋」，中部区域小字「<标题关键词 3-5 字>」（避开底部 15% 遮挡区）`
5. **负面提示**（固定文本，原样拼接）：
   `画面主体不要出现任何文字、字母、汉字，水印、logo、签名`

## 文章主题 → 视觉风格映射

| 文章主题 | 视觉风格 | 配色 / 质感关键 |
| --- | --- | --- |
| 科技数码 | 微缩等距 | 深蓝紫渐变，flat composition，900:383 |
| AI 知识科普 | Bento 几何信息图 | 蓝青 + 霓虹线条，clean typography，无背景杂物 |
| AI 历史人文 | 硅基复古版画 | 做旧纸张 + 古典蚀刻 + 霓虹电路 |
| AI 工具实战 | 数字质感物相 | 玻璃晶体 + 光影对比，macro shot，极浅景深 |
| 零售/商业运营 | 毛玻璃商务流 | Glassmorphism，蓝绿渐变，中央卡片 |
| SOP/教程 | 分步路径图 | 发光数字节点，图标，干净背景 |
| 职场/个人成长 | 虚拟实体质感 | 3D 极简物体特写，实体光影 |
| 文艺生活 | 胶片质感 | 柔焦颗粒，clean text overlay |
| 美食探店 | 食物特写 | 食欲感，text clearly focused |

## 文中配图：3 张按故事弧线分别构造

| 序号 | 画面定位 | 元素 |
| --- | --- | --- |
| 1 | **痛点** | 文章开头的困境/焦虑/失败具象化 |
| 2 | **转折** | 解决方案出现的关键时刻/工具 |
| 3 | **升华** | 文章结尾的理想状态/成果 |

3 张 prompt 必须互不相同，禁止复用。

## 禁止使用（看到一律按五要素重写）

- ❌ `科技风格插画`
- ❌ `为文章《XXX》生成封面`
- ❌ `AI 提升工作效率主题插画`
- ❌ `温暖治愈风格的封面`
- ❌ 任何只含"风格 + 主题"两要素的描述
